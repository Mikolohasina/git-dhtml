#  **DHTML Interactive Tools – Mini Applications Web**

##  **Fonctionnalités principales**

###  **1. Calcul automatique de produits**

* Tableau dynamique
* Calcul du coût total en temps réel
* Mise à jour automatique des valeurs

###  **2. Graphique interactif**

* Génération simple d’un graphique basé sur les données du tableau
* Visualisation intuitive des résultats

###  **3. Résolution de systèmes d’équations (Méthode de Cramer)**

* Système à 3 inconnues
* Calculs automatisés
* Affichage clair et détaillé des solutions

###  **4. Page de zoom d’images**

* Zoom fluide et interactif
* Navigation intuitive entre les visuels

###  **5. Section publicitaire dynamique**

* Affichage d’éléments animés ou interactifs
* Simulation de fonctionnalités web modernes

---

##  **Technologies utilisées**

* **HTML5**
* **CSS3**
* **JavaScript (DHTML)**
*  **Boostrap**
* **Jquery**
